# DMAAlarmTemplateCondition

| Item  | Format                                  | Description                                                            |

| Rules | Array of DMAAlarmTemplateConditionRule | See [DMAAlarmTemplateConditionRule](xref:DMAAlarmTemplateConditionRule). |
